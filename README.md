# 梅花心易

梅花易数随身起卦工具。时间 / 数字 / 随机三种起卦方式，自动排出本卦、互卦、变卦，标注动爻与体用，按五行生克给出断语。

## 本地运行

```bash
npm install
npm run dev
```

## 部署到 Vercel

1. 把本仓库推到 GitHub
2. 在 [vercel.com](https://vercel.com) 选择 **Add New → Project**，导入这个仓库
3. Vercel 会自动识别为 Vite 项目，直接点 **Deploy** 即可，无需任何配置

## 说明

- 时间起卦采用传统算法：年支数 + 农历月 + 农历日 起上卦，再加时辰数起下卦与动爻
- 农历月日需手动填写（手机日历可查），避免内置换算出错
- 断语依体用生克之大要，互卦观过程、变卦定结局，仅供参考自省
